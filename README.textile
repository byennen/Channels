h1. Channels

# Getting Started:

* Install gems
  bundle install

* Migrate database
  bundle exec rake db:migrate

* rake db:reset or rake db:seed

h2. S3 setup
* Make sure the following development environment variables are set /config/initializers/s3.rb:
  ENV['S3_UPLOADER_BUCKET']
  ENV['S3_UPLOADER_ACCESS_KEY']
  ENV['S3_UPLOADER_SECRET_ACCESS_KEY']
    
* Set your server host in vendor/assets/javascripts/s3_uploader/s3_uploader.js:
  var host = 'http://yourdomain.com';

* Deploy uploader javascript and form to s3:
  rake uploader:deploy
